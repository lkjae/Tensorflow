Old Machine + AI = Smart Machine

데이터로부터 학습하는 머신의 두뇌 -> 머신러닝
인간의 자연어를 이해하는 머신의 귀 -> 자연어 처리
물리적 행위를 제공하는 머신의 육체 -> 로보틱스
상황을 파악하고 이해하는 머신의 눈 -> 컴퓨터 비젼

분류: 0 또는 1, 참 거짓 등을 나누는 것
회귀: 여러 상황을 예측하여 실수값을 내는 것

 문제 영역에 잘 작동
